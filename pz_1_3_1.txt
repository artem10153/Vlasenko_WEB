git init  Эта команда используется для создания GIT репозитория.
git clone Команда git clone используется для клонирования репозитория
git add   Команда git add может быть использована для добавления файлов в индекс. 
git commit - m Команда git commit используется для коммита изменений в файлах проекта. Обратите внимание, что коммиты не сразу попадают на удаленный репозиторий.
